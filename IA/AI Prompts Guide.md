# AI Prompts Guide

A collection of effective prompt templates for using AI assistants in coding and research tasks.

---

## 📝 Prompts for Coding

When using AI for coding, be precise and provide context. The more details you give about the problem, the more relevant the response.

### General Principles
- ❌ Avoid: Asking for generic code
- ✅ Prefer: Specifying functionality, programming language, or algorithm

### Prompt Templates

#### 🔧 Debugging Help
```
I'm getting a [ERROR_MESSAGE] in my [PROGRAMMING_LANGUAGE] code. 
I'm trying to [TASK], but I'm not sure why it's failing. 
Here's the code: [INSERT_CODE]. 
Can you help me identify the issue?
```

#### ➕ Function Creation
```
Write a [PROGRAMMING_LANGUAGE] function that [DESCRIPTION_OF_TASK]. 
The function should [INCLUDE_SPECIFIC_CONDITIONS_OR_REQUIREMENTS].
```

#### ⚡ Code Optimization
```
Optimize this [PROGRAMMING_LANGUAGE] code that [DESCRIPTION_OF_TASK]. 
The current implementation is too [SLOW/INEFFICIENT] for [REASON].
```

#### 🌐 API Request Handling
```
Write a [PROGRAMMING_LANGUAGE] function that sends an API request 
to fetch [SPECIFIC_DATA] for a [SPECIFIC_TASK], handles the response, 
and [PROCESSES_THE_DATA].
```

#### ✅ Unit Test Creation
```
Write unit tests for a [PROGRAMMING_LANGUAGE] function that [TASK]. 
Include tests for both valid and invalid outputs to ensure the 
function handles edge cases.
```

## 🔬 Prompts for Research

AI can help with data analysis, trend identification, and summarizing findings. Frame your requests with clear scope and context.

### General Principles
- Specify the industry, topic, or data source
- Define the time period for analysis
- List specific aspects or sub-topics to focus on

### Prompt Templates

#### 📚 Literature Review Summary
```
Summarize key research findings from the last [TIME_PERIOD] on [SPECIFIC_TOPIC]. 
Focus on [SPECIFIC_ASPECTS], such as:
- [SUB_TOPIC_1]
- [SUB_TOPIC_2]
- [SUB_TOPIC_3]
```

#### 📊 Data Analysis Insights
```
Analyze the data for the last [TIME_PERIOD]. 
- Identify any trends or patterns
- Highlight which [PRODUCT_CATEGORY/SEGMENT] performed best
- Suggest potential reasons for [DECLINES/IMPROVEMENTS]
```

#### 📈 Market Research Summary
```
Provide a summary of recent market research on consumer behavior in [INDUSTRY]. 
Focus on [SPECIFIC_ASPECTS], such as:
- [CONSUMER_PREFERENCES]
- [BUYING_HABITS]
- [PRICING_SENSITIVITY]
```

#### 🏢 Competitive Analysis
```
Conduct a competitive analysis for a [NEW_PRODUCT_TYPE] entering the market. 
Identify:
- Main competitors
- Pricing strategies
- Key differentiators in the market
```

#### 📰 Industry Trends Report
```
Generate an industry trends report for [INDUSTRY] covering key developments 
in the last [TIME_PERIOD]. Include:
- Emerging technologies
- Market shifts
- Consumer behavior changes
- Regulatory updates
```

#### 📋 Survey Question Generation
```
Generate a list of [NUMBER] survey questions for [TARGET_AUDIENCE] to assess 
satisfaction with [SPECIFIC_PRODUCT/SERVICE]. 
The questions should focus on [SPECIFIC_ASPECTS] and include both 
qualitative and quantitative elements.
```

---

## 💡 Best Practices

1. **Be Specific**: Vague prompts yield vague answers
2. **Provide Context**: Include relevant background information
3. **Use Examples**: Show what you mean when possible
4. **Iterate**: Refine your prompts based on initial responses
5. **Verify**: Always validate AI-generated code and information
