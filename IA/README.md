## Quick Access

### Research Tools

| Tool | Purpose | Link |
|------|---------|------|
| **NotebookLM** | Summarize PDFs/articles | [notebooklm.google.com](https://notebooklm.google.com) |
| **Google AI Studio** | Advanced prompts with Gemini | [ai.google.dev](https://ai.google.dev) |
| **DeepSeek-R1** | Solve equations, code | [deepseek.com](https://deepseek.com) |
| **Perplexity AI** | Find papers, verify facts | [perplexity.ai](https://perplexity.ai) |
| **ChatPDF** | Interrogate PDFs | [chatpdf.com](https://chatpdf.com) |

## Recommended Workflow

1. **Research**: NotebookLM + Perplexity for literature review
2. **Simulation**: DeepSeek + Google AI Studio for coding
3. **Validation**: Grok + Qwen for theoretical questions
