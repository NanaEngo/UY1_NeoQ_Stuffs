## Quick Reference

### Essential Packages

```latex
\usepackage{amsmath, amssymb}    % Mathematics
\usepackage{graphicx}            % Figures
\usepackage{booktabs}            % Tables
\usepackage{hyperref}            % Links
\usepackage{siunitx}             % Units
\usepackage{tikz}                % Diagrams
```

### Common Issues

| Problem | Solution |
|---------|----------|
| Accents not working | Add `\usepackage[utf8]{inputenc}` |
| References show ?? | Compile 2-3 times |
| Image not found | Check path, use `\graphicspath` |
| Table too wide | Use `resizebox` or `tabular*` |
