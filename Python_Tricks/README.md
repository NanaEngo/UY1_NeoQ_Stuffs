## Quick Start

### 1. Install Miniforge

```bash
wget https://github.com/conda-forge/miniforge/releases/latest/download/Miniforge3-$(uname)-$(uname -m).sh
bash Miniforge3-$(uname)-$(uname -m).sh
```

### 2. Essential Packages

```bash
mamba install -c conda-forge jupyterlab numpy pandas matplotlib scikit-learn -y
```

### 3. Launch JupyterLab

```bash
jupyter lab
```
